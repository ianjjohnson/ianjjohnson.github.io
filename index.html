About Me

I am a computer science student and President's Scholar at Southern Methodist University with a passion for using computer science to solve real-world, non-CS related problems. I've been putting that passion to work since I was twelve, working on everything from data mining algorithms to RESTful database APIs to commercial scheduling software to mobile games. I'm currently pursuing both a Bachelors of Science and a Masters of Science in Computer Science, and plan to graduate with both degrees in 2019, after four total years at SMU.

Projects

These are some of the projects I've worked on over the last few years. Some are just fun side projects, some are more serious. The majority of my work is open-source and can be found on my GitHub page .

Network Topology Optimization Research

I researched how greedy algorithms for hierarchical LAN topology organization can be used to minimize electromagnetic interference in user-rich 802.11-based LANs. My team and I built a custom network modelling environment and implemented a number of distributed algorithms to build optimized LAN topologies. The research project was conducted under Dr. Engels of SMU as part of his Computer Networks and Distributed Systems Course. The paper is available here .

Full-Stack Bank Application

As part of a programming languages course at SMU, I built a full-stack mobile banking application using Java/Swing, PHP, and MySQL. The application features a multi-threaded front-end UI written with Swing, and a back-end API built with PHP and MySQL. The back-end includes security features to protect against SQL injection and cross-site scripting, and uses cryptographically secure salted hashes for user authentication. The first version of the back-end infrastructure is available on my GitHub page.

SHA-256 Partial Hash Collision Hunting

I built a parallelized Python program to pack-hunt partial SHA-256 hash collisions in an embarassingly parallel environment. I use the novel birthday-attack approach to generate 50-bit partial collisions. I presented research on the issue as part of my Discrete Computational Structures course.

High School Class Scheduling Software

I spent the better part of my years in high school working on a commercial-scale project which managed classroom, teacher, and student scheduling for a high school of 3600 students. The project was built around a proprietary depth-first-search-based machine learning algorithm. After years of development and deployment, the program built a complete schedule for CCHS for the 2015-2016 school year with a ten-fold decrease in scheduling conflicts from past schedules.

WikiBooks Search Engine

As part of a data structures course at SMU, I built a search engine in C++ which can parse a 1GB WikiBooks XML dump into an inverted file index in under 90 seconds and allows for lookup of individual words or multi-word queries using binary operators.

SSh over SMS - HackRice 2016

While attending my first ever hackathon, I built an SSh over SMS system featuring two-factor authentication over the course of 36 hours. The project was a finalist and won best first-time hack, and was featured by DevPost. A video demo, as well as the project source, can be found on DevPost.

Automatic Anti-Texting-and-Driving System

During my senior year of high school, I was hired to build an integrated Arduino-Android system which automatically detects any input activity on an Android device and sends a bluetooth signal to the Arduino which in turn can trigger a car alarm to dissuade use of cell phones while driving. I both designed and built the Android and Arduino apps and wired an Arduino circuitboard to receive Bluetooth signals and trigger the car alarm.

Motion Capture Pong Game

My first non-Hello-World project, which I wrote in 6th grade, was an implementation of pong in Processing which used motion capture from a computer's webcam to control paddle movement using hand motions of a user.

2D and 3D Game/Graphics System

Over the years, I've built a handful of games in Java and C++ using Unity, SDL and LWJGL. An example of a proof-of-concept for a Java game can be found on my GitHub page.

Classification by Association Algorithm in R

I implemented the CBA (Classification Based on Association rules) algorithm from Liu, et al. 1998 in R, with performance-critical data structures and algorithms implemented in C. I packaged the algorithm with a standard R interface, and the package is available on my GitHub page for download. The package is currently in the submission process to CRAN, the de-facto governing body of R packages.

RESTful API Backend for iOS Application

During my database course at SMU, I built a full-featured back-end RESTful API for a mobile application designed to help users find groups for pickup sports games. The RESTful API has custom session management using dynamic 256-byte session keys and user authentication and authorization. It provides an interface for users to join, leave, and create games, as well as maintain a friends list and chat with their friends or in their pickup game groups. I also served as the system administrator for the project, managing the servers and maintaining shell scripts to backup and re-configure servers to minimize back-end downtime.

Text Message Based Physics Assistant

Over the summer after my senior year of high school, I built a physics word problem solver and integrated it with a texting system that I built on my Linux server. The complete system can analyze physics word problems sent over text message and send a response with the answer(s) to the problem. A simplified but functioning proof-of-concept version can be found on my GitHub page.

ARM Assembly Sorting Algorithms

As part of an Assembly Language and Computer Organization course at SMU, I implemented various sorting algorithms in ARM assembly. The course required that we implement an in-place selection sort algorithm, but I expanded on the idea and implemented quicksort and mergesort as well.
